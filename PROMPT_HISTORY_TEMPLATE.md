## Prompt 2：建立後端
請用 Spring Boot 建立 REST API，包含 SQLite 資料表、CRUD 基礎功能，以及一個 ML 分析服務。

## Prompt 3：建立前端
請用 HTML/CSS/JS 做一個乾淨的介面，可以呼叫後端 API，新增資料並顯示分析結果。
